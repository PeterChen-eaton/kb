# Tampermonkey

This directory contains Tampermonkey userscripts for browser automation, website customization, and productivity enhancements.

## Available Scripts

*Scripts will be added here.*

## Categories

- **Website Customization**: Scripts to modify website appearance and behavior
- **Productivity**: Automation scripts for improved workflow
- **Utilities**: Helper scripts for common tasks
- **Enhancements**: Feature additions for various websites

## Usage Guidelines

1. **Install Tampermonkey**: Install the [Tampermonkey browser extension](https://www.tampermonkey.net/)
2. **Add Scripts**: Click on a script to view and install it
3. **Review Code**: Always review script content before installing
4. **Manage Scripts**: Use Tampermonkey dashboard to enable/disable scripts
5. **Keep Updated**: Check for script updates regularly

## Script Structure

Each Tampermonkey script typically includes:

```javascript
// ==UserScript==
// @name         Script Name
// @namespace    your-namespace
// @version      1.0
// @description  Script description
// @match        https://example.com/*
// @grant        none
// ==/UserScript==
```

## Contributing

When adding scripts to this directory:

1. **Document Thoroughly**: Include clear descriptions and usage instructions
2. **Specify Match Patterns**: Clearly indicate which websites the script affects
3. **Version Control**: Maintain version numbers and changelogs
4. **Test Thoroughly**: Ensure scripts work across different scenarios
5. **Security First**: Avoid storing sensitive data in scripts
